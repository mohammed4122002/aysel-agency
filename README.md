## Database Setup (Required)

This project stores dashboard content and consultation requests in **PostgreSQL** using **Prisma**.

1. Copy `.env.example` to `.env.local`
2. Add your database connection strings:
   - `DATABASE_URL` (pooled URL for runtime)
   - `DIRECT_URL` (direct URL for Prisma migrations)
3. Apply the schema:
   - `pnpm db:push`
4. Generate Prisma Client (if needed):
   - `pnpm db:generate`
5. (Optional) add `BLOB_READ_WRITE_TOKEN` for production media uploads.
